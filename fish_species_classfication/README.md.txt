 🐟 Fish Species Classification using YOLOv8-Tiny

This project detects and classifies different species of fish using the YOLOv8-tiny object detection model.

## 🚀 Steps to Run

### 1) Clone the Repository
```bash
git clone https://github.com/yourusername/fish-species-classification.git
cd fish-species-classification

2)Install Dependencies
pip install -r requirements.txt

3)Prepare Dataset
Download the Fish4Knowledge dataset from Roboflow
 in YOLOv8 format.
Extract it into the data/ folder.

4)Train the Model
python train.py

5)Test or Predict
python test.py
